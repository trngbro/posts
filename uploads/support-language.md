## What "Available" Means

When a language is marked as available:

- UI is translated
- Core pages work in that language
- Any documentation provided is in that language
- Translation is maintained over time

## Request a Language

If you want a language added, you can make a request via  [Pull Request Documentation](./pull-request-documentation). That’s all that’s needed—no social media followings or extra fuss.

## Contribute to Translations

Native speakers and bilingual contributors are welcome to help.

### How to Contribute

**Translators:**

1. Visit the translation folder
2. Start translating the strings for your language
3. Submit changes via pull request

**Reviewers:**

- Check translations for clarity
- Suggest minor improvements
- Ensure language reads naturally

### Translation Guidelines

- Keep original meaning
- Use a natural tone
- Adapt cultural references when necessary
- Be clear and concise

### Contributor Recognition

- Pull request credits
- Mention in documentation updates
- Acknowledgment in contributor list

## Quality Standards

Translations are checked for:

- Accuracy
- Readability
- Consistency
- Cultural appropriateness

## FAQ

**How do I change the website language?**  
Use the language selector in the settings.

**Why isn’t my language ready yet?**  
It depends on community demand and volunteer contributions. You can help by submitting a pull request.

**Can anyone contribute?**  
Yes, contributions are reviewed before being added.

**How long to add a new language?**  
Usually a few weeks to a couple of months, depending on the language and effort required.

**Is machine translation used?**  
Only as a helper—human review is always done.

**How to report translation issues?**  
Open a pull request or contact via email.
